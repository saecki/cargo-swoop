# cargo-swoop

A cli tool to find and remove target directories.

[![asciicast](https://asciinema.org/a/Nrd2FmjQXXAv84yKvkPH2GRZk.svg)](https://asciinema.org/a/Nrd2FmjQXXAv84yKvkPH2GRZk)

## Usage
```
Usage: cargo-swoop [OPTIONS] [SEARCH_DIR]

Arguments:
  [SEARCH_DIR]

Options:
      --follow-symlinks
      --show-empty
  -h, --help             Print help
```
